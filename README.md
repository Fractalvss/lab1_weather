# Лабораторная работа №1: Скрипт запроса данных из JSON API

## Описание
Bash-скрипт, который получает случайный факт о котах из API `catfact.ninja` и отображает его на веб-странице.

## Почему не погода?
API `wttr.in` и `open-meteo.com` недоступны (блокируются). 

Выбран `catfact.ninja` — стабильно работающий API с простым JSON-ответом.

## Возможности
- Получает случайный факт о котах через `curl`
- Парсит JSON с помощью `jq`
- Автоматически обновляет страницу каждую минуту через `cron`
- Отдаёт данные через `nginx`

## Установка
```bash
# Установить необходимые пакеты
sudo apt install nginx jq curl

# Скопировать скрипт
cp weather.sh ~/
chmod +x ~/weather.sh

# Добавить в cron
crontab -e
# Добавить строку: * * * * * /home/vboxuser/weather.sh >> /home/vboxuser/weather.log 2>&1
